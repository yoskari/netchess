# Netchess
Chess engine made with python and pygame.

## Requirements
* python3
* pygame

## Features
* regular chess movement
* score
* shows which tiles can be moved to

### Singleplayer
```
python singleplayer.py
```

### Server
```
python server.py
```

### Client
```
python client.py
```
